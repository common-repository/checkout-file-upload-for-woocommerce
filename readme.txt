== Features ==
- **Easy to use**
- **Image Preview**: Show Thumbnail for images
- **Attach to email**
- **Custom for each product**
- **File Type Validation**
- **File Size Validation**
- **Limit number of files Upload**
- **Send files as email attachment or as a links**
- **Support multiple languages**
- **Mobile Responsive**


== Installation ==
**Normal installation**

1. Download the checkout-upload-field.zip file to your computer.
2. Unzip the file.
3. Upload the `checkout-upload-field` directory to your `/wp-content/plugins/` directory.
4. Activate the plugin through the 'Plugins' menu in WordPress.
